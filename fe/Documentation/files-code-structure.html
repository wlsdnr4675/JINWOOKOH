Files & Code Structure

Root Structure

Path Description
/node_modules Yarn/NPM package installation folder
/package.json Yarn/NPM package manager config file
/public The root directory.
/src Source folder. Contains all components, pages, images, data and styles

Public Folder Structure

Path Description
public/js/custom Bootsnav plugin, 404 and Coming Soon page effects
public/js/portfolio Initialize CubePortfolio plugin and set properties.
public/js/revolution Contains all Revolution Slider plugin files.
public/jquery.min.js jQuery file
public/master.js Main jQuery file. Contains all functions for sliders, popup and parallax.
public/plugins.js Contains all jQuery plugins.

In public/js folder you can change settings for sliders, portfolio and all jQuery plugins. Also here you can add new plugins or custom jQuery code.
Here you find a list with all jQuery plugins we used in Anno theme.

Source Folder Structure

Path Description
src/assets Contains styles, fonts, images and videos. Main css file is "master.css". Imports all css files.
src/components Contains all components.
src/data Contains all data in json files. All files are structured by folders.
src/helpers Contains all theme helper functions.
src/pages Contains all pages.
