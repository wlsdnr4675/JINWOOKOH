Overview

Anno is One Page and Multipurpose React Template retina ready. It is amazing, professional and easy to use. You can use it for business, Photo Studio, Startup, Portfolio Template, Personal, Creative Agency, Corporate, Parallax, Mobile application and much more.

Anno React theme has a lot of components, and some of them are using jQuery. All sliders are animated by Revolution Slider. Also we use jQuery for popups, navbar, sidebar, portfolio and parallax.

Main features and components of our theme are:

  • + 10+ home page demos including one page home page.
  • + 20+ HTML pages.
  • + React and Jquery combined.
  • + Bootstrap 3x responsive.
  • + Beautiful and gorgeous header styles like transparent, white, dark, top bar, center navigation, gradient navigation, centered logo, top logo, right menu and centered navigation.
  • + Image and video background.
  • + Ionicons pack
  • + Revolution slider (premium jQuery plugin worth $16 included free of cost).
  • + CubePortfolio (16$ - jQuery plugin)
  • + Touch slick slider.
  • + Sticky header menus.
  • + Smooth parallax scrolling sections.
  • + Video in background and popup.
  • + Animated Text.

  • + Smooth scroll with one page home page ready demo.
  • + Team members and clients grid and carousel.
