Specials components

Header

Header component who contain SearchOverlay component, needs to import the helper HandleSearchOverlay from helpers folder, then get the props and functions from helper. This will be passed to SearchOverlay component, namely AttrNav component.


  import React from "react";
  import HandleSearchOverlay from "../../helpers/HandleSearchOverlay";
  import SideNav from "./SideNav";
  import SearchOverlay from "./SearchOverlay";
  import NavbarTwo from "./NavbarTwo";
  import NavbarHeader from "./NavbarHeader/NavbarHeader";
  import AttrNav from "./AttrNav";
                
  const HeaderThree = ({ type, data }) => {
    const { show, showSearchForm, hideSearchForm } = HandleSearchOverlay();
      return (
        <nav className="navbar navbar-default navbar-fixed navbar-transparent white bootsnav on no-full brand-center menu-center">
          <SearchOverlay show={show} onClick={hideSearchForm} />
          <div className={"container" + (type === "wide" ? "-fluid" : "")}>
            <AttrNav border showSearchForm={showSearchForm} />
            <NavbarHeader />
            <NavbarTwo data={data} />
          </div>
          <SideNav data={data} />
        </nav>
    );
  };
                
  export default HeaderThree;

Footer

Footer component use LoadScript helper. Basically, Footer loads the master.js file, who contain almost all jQuery functions for sliders, popups and parallax. 


  import React from "react";
  import LoadScript from "../../helpers/LoadScript";
                              
  const Footer = ({ bg }) => {
    LoadScript("./js/master.js");
      return (
           ...
      );
  };
                              
  export default Footer;

Creating a page without footer, require to import the LoadScript helper and load master.js file, located in public/js/master.js. Do not use LoadScript in useEffect. LoadScript has an useEffect hook by itself.

Portfolio

Portfolio component uses Cube Portfolio jQuery plugin. It requires to import LoadScript helper then load protfolio-grid.js from public/js/portfolio/portfolio-grid.js. In Portfolio component you can see the right path and how it's loaded.
