Components structure

For Anno was developed about 100+ components. Every section of a page is a component and have its custom props and functionality. The components are reusable and can be customized as well. One of them needs a special approach when they are implemented. They are mentioned in Specials section.

All components are located in component folder. All files are well structured, contains clean code and description where necessary. Here you cand find all props and details of main components.
