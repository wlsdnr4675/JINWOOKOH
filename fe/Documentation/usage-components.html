Usage components

Suppose we create a new page: 


  import imgAbout from "../../assets/images/bg-left-img.jpg";
  import AboutOne from "../../components/About/AboutOne";
                
  const CustomPage = () => (
      <AboutOne tagline="About Us" title="Partners for Growth" image={imgAbout}>
          The process of improving some measure of an enterprise's success.
          Business growth can be achieved either by boosting the top line or
          revenue of the business with greater product sales or service income, or
          by increasing the bottom line or profitability of the operation by
          minimizing costs.
      </AboutOne> 
  );

export default CustomPage;

All components are made to be reusable without having to duplicate the code. We created props for every custom element in the component, even for images or backfont.

Here you cand find a list with all props and details of main components.

To prevent errors you need to pay attention on component props. Some components have props that are required.
